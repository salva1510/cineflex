CINEFLEX BUILD 5.0 — SMART ASSISTANT + DISCOVERY 2.0

INSTALL
1. Replace all existing website files with this package.
2. Clear browser site data or reinstall the CineFlex PWA.
3. Open the cyan magic-wand button above the bottom navigation.

EXAMPLE REQUESTS
- Korean action series
- Nakakatawang family movie
- Anime na may magandang story
- Parang John Wick
- Hidden gems
- Top-rated horror movie 2025

HOW IT WORKS
This version uses an on-device intent parser plus TMDB search/discover/recommendation APIs. It does not send user prompts to a paid external AI service. Recent prompts and taste signals stay in localStorage and are used for the Smart Discovery row.

PRESERVED
Authentication, profiles, Watch Party, chat, progression, Admin Studio, moderation, requests, reports, notifications, and the existing player are preserved.

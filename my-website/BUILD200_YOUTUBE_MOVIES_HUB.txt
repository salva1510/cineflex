CINEFLEX BUILD 20.0 — YOUTUBE MOVIES HUB

Added:
- Dedicated YouTube Movies hub inside the menu drawer
- Official VIVA Films channel integration through YouTube Data API v3
- Full-movie filtering (minimum 60 minutes + full-movie title/description signal)
- Embeddable/public status check
- Action, Comedy, Romance, Horror and Drama filters
- Trending carousel using available YouTube view counts
- New on YouTube badge for uploads from the last 14 days
- Search, sorting, favorites, previous/next and fullscreen player
- 6-hour local catalog cache to reduce API quota usage
- youtube-nocookie.com privacy-enhanced embed player
- Safe API-key placeholder in js/modules/youtube-movies-config.js

Footer:
- All Movies remains in footer navigation
- Music remains removed from footer navigation
- Music Hub remains available in the menu drawer

SETUP REQUIRED:
1. Revoke the API key previously shared in chat.
2. Create a NEW browser-restricted YouTube Data API key.
3. Restrict websites to:
   https://cineflex.online/*
   https://www.cineflex.online/*
4. Restrict API to YouTube Data API v3 only.
5. Open js/modules/youtube-movies-config.js and replace:
   YOUR_RESTRICTED_YOUTUBE_API_KEY
